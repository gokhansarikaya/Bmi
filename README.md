# BMI
The body mass index (BMI) is a value derived from the mass (weight) and height of an individual.

The body mass index (BMI) is a value derived from the mass (weight) and height of an individual.

  *  BMI < 18.5 Underweight
  *  18.5 <= BMI <= 24.9 Normal
  *  25 <= BMI <= 29.9 Overweight
  *  BMI >= 30.0 Obese
